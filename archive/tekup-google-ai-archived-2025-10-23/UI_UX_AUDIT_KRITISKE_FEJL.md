### Rate Limiter Konfiguration
**Fil:** `src/middleware/rateLimiter.ts`


**Problem:**
- Dashboard laver burst af 7-10 requests ved f√∏rste load

**Fil:** `client/src/pages/Leads/Leads.tsx` (lines 44-66)

```typescript

  if (!acc.seen.has(key)) {
    acc.seen.add(key);
    acc.leads.push(lead);
  }
  return acc;

```

**Problem:** Deduplication logic k√∏res i frontend, men backend burde ikke sende duplicates!

**Backend Issue:** Leadmail.no parser eller database insert logic tillader duplicate entries.
