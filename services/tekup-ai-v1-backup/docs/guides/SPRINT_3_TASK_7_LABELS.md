State Machine Pattern\n\n\n\n**Why:** Prevents invalid state transitions\n\n\n\n- Can't go from `new_lead` → `completed` (must go through workflow)\n\n- Validates transitions before applying labels\n\n- Clear, documented workflow\n\n\n\n### 2.
